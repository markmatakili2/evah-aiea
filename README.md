# AI Epilepsy Assistant (AIEA)
## Clinical Decision Support Prototype

This project is a dedicated prototype for the AI Epilepsy Assistant (AIEA), designed to empower Community Health Workers (CHWs) and Clinicians in rural and remote settings with WHO mhGAP-aligned management tools.

### Core Features
- **Guided Clinical Encounters**: Step-by-step assessment based on international epilepsy protocols.
- **AI Clinical Engine**: Suggestive analysis for triage, urgency classification, and anti-stigma counseling.
- **Regional Registry**: Centralized patient history and referral tracking for care coordination.
- **Supervisor Oversight**: Administrative tools for facility onboarding and healthcare worker approval.

### Ethics & Governance
AIEA is built with a focus on clinical safety and data privacy. Final decision authority remains with the healthcare professional. All AI suggestions are clearly labeled as suggestive only.
